# Pond

Pond is a lo-fi productivity & relax platform. It includes a habit tracker, a clock, an automatic screensaver mode, a timer interface, a music player with OST, a shop, and a pond management interface.

The centerpiece and background of the application is a 2D pond, onto which fish flow to and from, and being present on screen, they exhibit unique behaviors and interactions.

## About This Project

This is a Godot 4 project emphasizing ecological simulation and ambient productivity support. Fish species (Guppies, Sabalos, Dientudos) interact naturally through feeding and predation, independent of any competitive win conditions. The pond runs continuously in the background, even when the application is minimized, making it a persistent desktop companion.

For detailed product vision, architecture philosophy, and development criteria, see [`.github/instructions/product-vision.instructions.md`](.github/instructions/product-vision.instructions.md).
